Object Detection and Visual Question Answering

Object Detection

Object detection is a computer vision technique that allows us to identify and locate objects in an image or video. With this kind of identification and localization, object detection can be used to count objects in a scene and determine and track their precise locations, all while accurately labeling them.

Models Used

We tried multiple Object Detection models like Yolov3, Yolov4, Yolov5, Detectron2 and SSD. After testing these models we decided to go with Yolov4 and Yolov4-tiny for it's performance and speed.

  • For object detection on image we used Yolov4 model trained on COCO dataset and custom dataset
  • For object detection on video we used Yolov4-tiny model trained on COCO dataset and custom dataset. Compared to Yolov4, Yolov4-tiny is much faster on video with good detection.

Visual Question Answering

The Visual Question Answering (VQA) task combines challenges for processing data with both Visual and Linguistic processing, to answer basic `common sense' questions about given images. Given an image and a question in natural language, the VQA system tries to find the correct answer to it.

Models Used

We trained Visual Questioning Answering model on COCO Image Dataset and their Annotations

  • We trained the model by training custom CNN model on COCO images and we used LSTM for the questions extracted from the annotations
  • Then we combined both of this model to get out final VQA model
